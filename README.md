# Nydo Project

Private working repository for Nydo product, brand, and planning materials.

Nydo is a pre-launch, compatibility-based flatmate matching platform for international students at top European universities. The project is centred on helping students find verified, compatible flatmates before housing decisions become expensive and stressful.

Public tagline: "Home starts with who, not where."

## What Is Here

- `AGENTS.md`: core project, voice, positioning, and writing guidance.
- `DESIGN.md`: design system, layout, motion, imagery, and quality guidance.
- `memory/`: workspace memory rules and project context for future sessions.
- `tasks/`: active sprint notes and project setup tasks.
- `prd-draft (1).md`: reusable PRD drafting workflow for AI-enabled app builders.

## Handling Sensitive Information

Do not commit API keys, passwords, `.env` files, private keys, customer data, or partner credentials. Use `.env.example` for placeholder configuration only.
